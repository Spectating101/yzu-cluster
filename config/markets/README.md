# Market Universes (Yahoo Finance tickers)

These ticker lists are used to build tidy panels via:

`python3 Sharpe-Renaissance/scripts/build_market_panels.py --market all --period 10y`

Notes:
- These are **proxies** (mostly US-listed ETFs/ADRs) so we can test quickly with yfinance data.
- You can swap/add true local tickers later (e.g. `2330.TW`) if your broker/data supports them.
