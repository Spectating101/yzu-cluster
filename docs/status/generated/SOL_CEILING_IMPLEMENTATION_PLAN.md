# Sol ceiling implementation branch

This branch is reserved for an adversarial presentation-layer reconstruction of Research Drive.

## Authority

- preserve RC2.1 interaction stability, compactness, and release discipline;
- preserve RC3 research semantics, evidence gaps, lifecycle truth, Composer, and live backend integration;
- rebuild only the visible composition that prevents those capabilities from reading as one governed research control plane;
- retain `agent/rc3-semantic-research-desk` as the rollback baseline.

## Visual thesis

A calm institutional research instrument: part evidence archive, part analytical workbench, part controlled execution system.

The interface must not read as a generic AI dashboard, file manager, cloud console, or chatbot wrapped in cards.

## Acceptance target

1. One dominant research object per page.
2. No duplicated centre/rail prose.
3. Faculty-facing value before implementation metrics.
4. Exact backend truth retained.
5. Distinctive evidence-lineage and research-state grammar.
6. Desktop-first exact-SHA capture before any merge decision.

## Page anchors

- Home: research brief — continue, decide, newly available, unblocked.
- Library: controlled evidence estate — readiness, grain, coverage, provenance, reuse.
- Discover: uncontrolled evidence space — held, connected, missing, acquisition routes.
- Synthesis: flagship Research Construction workbench.
- Resources: faculty trust surface with operator detail subordinate.
