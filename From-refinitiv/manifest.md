The "Survivorship Bias" ProtocolStandard datasets only include companies currently listed in the S&P 500. This introduces bias (winners only).Our Solution: The pipeline iteratively queried 0#.SPX(YYYYMMDD) for every January 15th from 2015 to 2025.Result: The dataset includes "Graveyard" companies (e.g., Silicon Valley Bank, First Republic, Signature Bank) that were delisted. This allows the AI to learn "Bankruptcy Risk."B. The "Volatility Surface" LogicTo price options and predict crashes, Price (TR.PriceClose) is insufficient. We harvested the "Smile":TR.ImpVolDelta25: The cost of Upside Calls (Greed).TR.ImpVolPutDelta25: The cost of Downside Puts (Fear).Skew Calculation: PutDelta25 - Delta25 provides the "Crash Probability" metric used by hedge funds.C. Data Cleaning & AlignmentDuplicate Handling: Refinitiv API occasionally returns duplicate timestamps for holidays. A post-processing script (df[~df.index.duplicated]) sanitized the index.Calendar Separation: Assets with different trading calendars (Crypto 24/7 vs. TSMC Mon-Fri) were split into separate files (PATCH_...csv) to prevent NaN injection during merging.3.
